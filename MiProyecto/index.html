Git & GitHub Sitio de ejemplo

Git es un sistema de control de versiones que te permite gestionar y hacer seguimiento de los cambios realizados en el código fuente de un proyecto. Es ideal para trabajar de manera colaborativa, ya que permite que múltiples personas trabajen en el mismo proyecto sin sobrescribir el trabajo de los demás.

GitHub es una plataforma en línea que permite almacenar proyectos Git en la nube. No solo es útil para respaldar tu código, sino también para colaborar con otros desarrolladores de forma remota.

Pasos básicos para usar Git y GitHub

  1. Inicializar un repositorio Git:

    2. En la terminal, navega a tu carpeta del proyecto y ejecuta el siguiente comando para inicializar un repositorio local:

    git init
  2. Agregar archivos al área de preparación:

    3. Después de hacer cambios en tus archivos, debes prepararlos para el commit con:

    git add .
  3. Realizar un commit:

    4. El commit guarda una "instantánea" de tu proyecto. Escribe un mensaje claro describiendo los cambios:

    git commit -m "Mensaje descriptivo"
  4. Crear un repositorio en GitHub:

    5. Ve a GitHub y crea un nuevo repositorio. Copia la URL del repositorio.

  5. Vincular el repositorio local con GitHub:

    6. En la terminal, vincula tu repositorio local con el remoto en GitHub usando el siguiente comando:

    git remote add origin https://github.com/tu-usuario/tu-repositorio.git
  6. Subir tu proyecto a GitHub:

    7. Finalmente, sube tu proyecto con:

    git push -u origin master

¡Y listo! Ahora nuestro proyecto está disponible en GitHub y podemos seguir haciendo cambios, comitearlos y enviarlos con git push.
